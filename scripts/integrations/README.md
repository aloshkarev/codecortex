# MCP Integration Snippets

This folder contains ready-to-copy snippets for AI clients:

- `cursor.mcp.json` -> Cursor MCP config
- `zed.settings.json` -> Zed `context_servers` config block
- `gemini.settings.json` -> Gemini CLI `settings.json` block

CLI one-line commands for other clients:

- Claude Code:
  - `claude mcp add cortex -- cortex mcp start`
- Codex CLI:
  - `codex mcp add cortex -- cortex mcp start`

Bootstrap script:

- `../bootstrap-codecortex-mcp.sh <repo-path>`
